Quotify Guest Management App – React (with Refs)

A simple guest management application built using React (without useState).
This project demonstrates how to use Refs, forms, and table rendering in React while handling:

1. Adding guest entries

2. Checking duplicate guest names

3. Validating guest count

4. Displaying check-in time using Date()

5. Updating available seats

6. Removing entries and restoring seat count
This project is ideal for beginners learning Refs, DOM access, and basic React rendering logic.

Features
1. Add guests with name, count, and phone number

2. Automatically generate check-in timestamp

3. Prevent duplicate names

4. Prevent exceeding total seating capacity

5. Remove entries and restore seat count

6. Display all records in a table

7. Simple and clean UI


Technologies Used:-
-HTML
-CSS
-JavaScript
-React (via CDN scripts)

How to Run:-

-Download or clone the repository

-Open index.html in any web browser

-The app will run instantly — no build tools required
